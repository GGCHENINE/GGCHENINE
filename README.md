# GGCHENINE Learning Projects

This repository contains my Python learning projects for data analysis and risk control.

## Main Contents

- `home-credit-default-risk/`
  A beginner-friendly Home Credit Default Risk analysis project focused on data understanding, EDA, grouped default-rate analysis, and risk interpretation.
- `py_for_da_analysis/`
  Practice code and notes from Python for Data Analysis.
- `python_for_risk_control_1/`
  Risk control learning exercises and examples.
- `python_for_risk_control_2/`
  Additional risk control learning exercises and examples.
