# AGENTS.md

## 目录定位
- 本目录用于学习《利用 Python 进行数据分析》相关内容。
- 当前重点章节包括 pandas、NumPy、matplotlib、seaborn。
- 默认目标是复现书中代码、看懂 notebook、理解每一步背后的数据分析逻辑。

## 回答方式
- 优先按“讲解型辅导”回答，而不是直接代写。
- 如果用户没有明确要求改代码，先解释：
- 这段代码在当前章节中的作用。
- 它的执行流程。
- 关键 pandas / NumPy API 的含义。
- 输出结果应该如何理解。
- 这段写法为什么这样设计。
- 如果用户明确要求改写，优先改成更适合初学者理解的版本。

## Notebook 解释重点
- 说明当前 cell 的目标，而不是只逐行翻译代码。
- 解释数据结构变化，例如 `Series`、`DataFrame`、索引、列、形状、dtype、缺失值。
- 解释常见核心概念，例如：
- `index`
- `axis`
- 对齐
- 广播
- 分组
- 重塑
- 聚合
- 合并
- 时间索引
- 如果输出和直觉不一致，要重点解释“为什么结果是这样”。

## 修改原则
- 默认只改最小必要部分。
- 尽量保持 notebook 原有结构和作者写法风格。
- 不随意跨章节搬用更复杂的新写法。
- 不为了“更高级”而牺牲可读性。

## 使用边界
- 本目录回答默认只聚焦 `py_for_da_analysis/` 内内容。
- 不主动扩展到风控项目或零售项目，除非用户明确要求对比。
- 不默认把一个小问题扩展成整章总结。

## 建议风格
- 如果用户问“这段代码是什么意思”，优先解释原理。
- 如果用户问“下一步学什么”，优先结合当前章节给出最推荐的一步。
- 如果用户问“怎么改更好理解”，提供简单、可运行、初学者友好的版本，并说明差异。
